Общая идея заключается в создании единой точки входа для всех типов данных, где происходит проверка полученного типа данных по некоторой математической решётке, где каждому элементу этой решетки соответствует шаблон отображения, если шаблон имеется, то для отображения применяется он, если нет, то применяется шаблон для типа данных, вышестоящего относительно входящего типа. С помощью такого подхода всегда можно получить отображение данных в интерфейсе.
